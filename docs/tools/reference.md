# Reference

Quick reference for color codes, border styles, font names, and the AARRGGBB color format.

## Color Format (AARRGGBB)

Colors are specified in 8-character hex format: **AARRGGBB**

| Component | Meaning | Range |
|-----------|---------|-------|
| **AA** | Alpha / Opacity | `FF` = opaque |
| **RR** | Red component | `00`–`FF` |
| **GG** | Green component | `00`–`FF` |
| **BB** | Blue component | `00`–`FF` |

> **Note:** Some tools also accept 6-character hex (RRGGBB) without the alpha channel.

## Common Colors

| Color | Hex (AARRGGBB) | Hex (RRGGBB) |
|-------|----------------|--------------|
| Black | `FF000000` | `000000` |
| White | `FFFFFFFF` | `FFFFFF` |
| Red | `FFFF0000` | `FF0000` |
| Green | `FF00FF00` | `00FF00` / `00B050` / `006100` |
| Blue | `FF0000FF` | `0000FF` / `4472C4` / `1F4E78` |
| Yellow | — | `FFC000` |
| Orange | — | `FF9900` / `ED7D31` |
| Gray | `FFD3D3D3` | `E7E6E6` / `D9D9D9` |
| Light Gray | `FFF9F9F9` | — |
| Dark Blue | `FF1F4E78` | `1F4E78` |
